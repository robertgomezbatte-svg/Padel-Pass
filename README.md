# Padel Pass (MVP)

Web estática estilo serio/gaming para el proyecto **Padel Pass**.

## Qué incluye
- Inicio: próximos eventos, top jugadores, objetivos mensuales (demo)
- Pase: 30 niveles, progresión y recompensas
- Eventos: listado con búsqueda
- Jugadores: ranking con filtros
- Perfil: stats + progreso + historial de partidos

## Cómo ejecutarlo en local
Abrir `index.html` directamente o usar un servidor simple:

### Opción A: VS Code Live Server
Instala la extensión y abre el proyecto.

### Opción B: Python
```bash
python -m http.server 8080
